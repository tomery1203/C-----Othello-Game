# Othello Game

This is a mini game made in C# that allows you to play against another player or against the computer. You can also choose the size of the board.

#Installation

To use this project, download and build the source code.

#Usage
To play the Othello game, follow these steps:

1.Choose who you want to play against (another player or the computer).
2.Choose the size of the board.
3.Start playing!


#Features
The Othello game includes the following features:

1.Play against another player or against the computer.
2.Choose the size of the board.
3.Simple and intuitive interface.

#Technologies

This project was created using C# and the .NET Framework.
